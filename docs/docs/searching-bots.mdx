# Searching for Bots and Apps
You can search for a specific bot by typing in keywords and hitting the “Search” button to view results.

![Searching for bots on Mezon Top Board](/img/explore.png)

## Using Filters and Sorting
When searching for bots and apps, you can also refine your search with filters and sorting. You can choose multiple filters for more accurate results. For sorting, please click on the dropdown menu.

![Using filters and sorting options](/img/filter.png)
