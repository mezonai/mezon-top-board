# Welcome to the Mezon Top Board Guide!
This guide is designed to provide answers to commonly asked questions about how to use and navigate Mezon Top Board. Whether you're a new user or an experienced user looking for more information, this guide has you covered. If you can't find the answer you're looking for, please don't hesitate to contact our support team for assistance.

## What is Mezon Top Board?
Mezon Top Board is a platform designed for the Mezon user community, aimed at helping users discover, review, and install bots, tools, and extensions for their Mezon servers. Inspired by platforms like Discord Bots and Discord Apps, Mezon Top Board enables users to explore and interact with a wide range of bots, applications, and services that enhance their Mezon experience.
